TCP状态查看：

netstat -n | grep 60.214.137.170

查看公网IP：ip addr show eth0 | grep inet | awk ‘{ print $2; }’ | sed ‘s/\/.*$//‘

查看系统：

lsb_release -a
cat /proc/version

查看yum安装创建相关文件：

rpm -qa|grep redis
rpm -ql redis
